# SCT_ML_02_Customer_Segmentation
Customer Segmentation using K-Means Clustering on the Mall Customers Dataset. This project groups customers based on annual income and spending score to identify distinct customer segments.
# SCT_ML_02_Customer_Segmentation

## Task Overview

This project is completed as part of the SkillCraft Technology Machine Learning Internship Task 02.

The objective is to create a K-Means Clustering algorithm to group customers of a retail store based on their purchase behavior using the Mall Customer Dataset.

## Dataset

* Mall Customer Dataset
* Features Used:

  * Annual Income (k$)
  * Spending Score (1-100)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Project Steps

1. Load and explore the dataset.
2. Select relevant features for clustering.
3. Standardize the data using StandardScaler.
4. Use the Elbow Method to determine the optimal number of clusters.
5. Apply K-Means Clustering.
6. Visualize customer segments using scatter plots.
7. Analyze the resulting customer groups.

## Results

* Successfully identified customer segments using K-Means clustering.
* Determined the optimal number of clusters using the Elbow Method.
* Visualized customer groups based on Annual Income and Spending Score.
